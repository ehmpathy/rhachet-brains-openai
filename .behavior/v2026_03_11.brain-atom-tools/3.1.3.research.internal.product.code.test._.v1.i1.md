## patterns identified

### [REUSE] given-when-then BDD structure

> "given('[case1] genBrainAtom({ slug: "openai/gpt/4o-mini" })', () => {"
> "when('[t0] inspecting the atom', () => {"
> "then('repo is \"openai\"', () => {" [1]

**reuse**: same BDD structure for tool use tests. cases map to criteria:
- `[case1]` tools plugged, brain invokes tools
- `[case2]` tools plugged, brain answers directly
- `[case3]` no tools plugged, extant behavior
- `[case4]` episode continuation with tool results
- `[case5]` multiple tool invocations
