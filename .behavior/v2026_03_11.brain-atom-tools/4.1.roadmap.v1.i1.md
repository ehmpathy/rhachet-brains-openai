### depends on

### read before start

- `.behavior/v2026_03_11.brain-atom-tools/3.3.1.blueprint.product.v1.i1.md` (openai map section)

### checklist

- [ ] create `src/infra/cast/castIntoOpenaiFunctionTool.ts`
  - transform `BrainPlugToolDefinition` → openai function tool
  - `type: 'function'`
  - `name: tool.slug`
  - `description: tool.description`
  - `parameters: zodToJsonSchema(tool.schema.input)`
  - `strict: true`

- [ ] create `src/infra/cast/castIntoOpenaiFunctionTool.test.ts`
  - test: slug maps to name
  - test: description maps to description
  - test: zod schema maps to JSON schema

- [ ] create `src/infra/cast/castIntoOpenaiFunctionCallOutput.ts`
  - transform `BrainPlugToolExecution` → openai function_call_output
  - `type: 'function_call_output'`
  - `call_id: exec.exid`
  - `output: JSON.stringify(exec.output)`

- [ ] create `src/infra/cast/castIntoOpenaiFunctionCallOutput.test.ts`
  - test: exid maps to call_id
  - test: output is JSON stringified
  - test: success signal handled
  - test: error signal handled

- [ ] create `src/infra/cast/castFromOpenaiFunctionCall.ts`
  - transform openai function_call → `BrainPlugToolInvocation`
  - `exid: item.call_id`
  - `slug: item.name`
  - `input: JSON.parse(item.arguments)`

- [ ] create `src/infra/cast/castFromOpenaiFunctionCall.test.ts`
  - test: call_id maps to exid
  - test: name maps to slug
  - test: arguments JSON parsed to input

